## ğŸš€ Getting Started

### **1. Open the Application**
Simply open `index.html` in a modern web browser.

### **2. Fund Your Wallet**
- Click the "Wallet" button in the top bar
- Or navigate to the Wallet tab
- Click "Fund Wallet"
- Enter amount and select payment method

### **3. Buy Data**
- Tap on a network card (MTN, Telecel, or AirtelTigo)
- Select a bundle size
- Enter phone number
- Confirm purchase

### **4. Access Admin Panel**
Open `admin.html` to view:
- Transaction history
- Revenue statistics
- Network management
